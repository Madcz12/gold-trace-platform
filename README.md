Plataforma para el seguimiento y custodia del oro en empresas dedicadas al rubro.

Pasos de la instalación: 

1. Tener ambos proyectos en la misma raiz (workspace recomendado)
2. Abrir terminal para el backend (server) y para el frontend (client)
3. npm install para instalar las dependencias
4. npm run migrate para cargar los datos en la base de datos
5. npm run dev
